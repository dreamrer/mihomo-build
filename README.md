# mihomo-build

Multi-platform build pipeline. Releases are published here automatically when a new version tag is pushed.

## Platforms

| Platform | Architecture |
|----------|-------------|
| Android  | arm64-v8a / armeabi-v7a |
| Windows  | x86_64 |
| Linux    | x86_64 |
| macOS    | arm64 (Apple Silicon) |
| macOS    | x86_64 (Intel) |
